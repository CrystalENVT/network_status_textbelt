# Network Status (via) Textbelt

This is a tool that pings a host & notifies when that host goes down or comes up via [Textbelt](https://textbelt.com/) (a pre-pay SMS API)

From the Textbelt Site:
> Textbelt is a no-nonsense SMS API built for developers who just want to send SMS
